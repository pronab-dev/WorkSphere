# WorkSphere

WorkSphere is a modern Human Resource Management System (HRMS) built with React, Laravel, and MySQL.

## Tech Stack

- Frontend: React + Vite
- Backend: Laravel
- Database: MySQL
- API: RESTful API

## Features

- Employee Management
- Department Management
- Attendance Management
- Leave Management
- Payroll
- Dashboard & Analytics
- Role-Based Access Control
- Authentication & Authorization

## Project Structure

```
WorkSphere/
├── worksphere-client/
└── worksphere-api/
```
